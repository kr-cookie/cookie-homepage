홈 담당 에이전트 인계

HAOS 복구 → 자비스 운영 설계 → 쿠키의 관리감독과 운영 지원

왜 중요한가

자비스가 Home Assistant를 직접 관제하더라도 쿠키는 상태, 권한, 위험 신호, 중지 기준을 함께 봐야 합니다.

현재 상태

현재는 HAOS 복구와 자비스 역할 분리를 준비하는 단계입니다. 자비스는 Home Assistant 관제를 맡고, 쿠키는 쌀떡과 소통하며 자비스의 결과를 관리감독합니다.

다음 행동

  • HAOS 재설치/복구 절차와 되돌림 기준 작성
  • 자비스의 보고선, 권한 요청, 위험 중지 기준 정의
  • 홈 제어·자동화 권한은 읽기, 제안, 실행 단계로 분리
