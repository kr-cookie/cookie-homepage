일정-작업공간 연결 흐름

촬영 일정 수집 → 캘린더 일정 관리 → 촬영 데이터 작업공간

왜 중요한가

촬영 일정이 들어오면 캘린더와 데이터 폴더 준비까지 이어져야 쌀떡의 현장 업무 부담이 줄어듭니다.

현재 상태

현재는 대표님 대화에서 촬영 일정을 안전하게 추출하고, 기존 캘린더 양식과 정리중 폴더 규칙으로 이어지게 만드는 단계입니다.

다음 행동

  • 대표님 대화창/DB 읽기 검증과 허용 대상 확인
  • 기존 캘린더 제목 규칙과 메모 누적 방식 유지
  • 정리중 폴더 생성은 기존 폴더 양식과 충돌 여부를 확인한 뒤 실행
